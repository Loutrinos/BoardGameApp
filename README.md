# Boardgamesson

Simple application that shows the game sessions we have in our group, the players that joined and more information about the game

## Getting started

This project doesn't have any dependencies, it's meant to be fast and easy to setup and make it work

Libraries used for this app:
- [MithilJS](https://mithril.js.org/)
- [SemanticUI](https://semantic-ui.com/)

Tools
- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
- [Live Sass Compiler](https://marketplace.visualstudio.com/items?itemName=ritwickdey.live-sass)
